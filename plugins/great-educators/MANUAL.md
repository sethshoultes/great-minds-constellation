# great-educators

## Manual Installation

1. Clone the constellation repo:
   ```bash
   git clone https://github.com/sethshoultes/great-minds-constellation.git
   ```

2. Install via Claude Code:
   ```
   /plugin add ./plugins/great-educators
   ```

3. Or install from the marketplace:
   ```
   /marketplace install great-educators
   ```

## Verification

After install, verify the plugin loaded:

```
/agency-setup
```

You should see three educator personas listed: Feynman, Montessori, Freire.
